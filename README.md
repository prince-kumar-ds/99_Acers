

# 🏡 99 Acers – Real Estate Price Prediction Platform

An end-to-end **Machine Learning project** that predicts real estate prices based on key features such as location, area (sqft), and BHK configuration. This project demonstrates the complete data science lifecycle — from data collection to deployment.

## 🚀 Project Overview

The goal of this project is to build a **robust real estate price prediction system** using real-world data. It covers:

- Data Collection (including Web Scraping)
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering & Selection
- Outlier Detection & Removal
- Model Building & Optimization
- Recommendation System
- Web App Deployment

---

## 📊 Key Features

### 🔹 Data Collection
- Extracted real estate data using web scraping techniques
- Combined multiple datasets for better generalization

### 🔹 Data Preprocessing
- Handled missing values
- Cleaned inconsistent and noisy data
- Standardized formats (area, price, locations)

### 🔹 Exploratory Data Analysis (EDA)
- Analyzed:
  - Location-wise pricing trends
  - Area vs Price relationships
  - BHK distribution
- Used visualizations for insights

### 🔹 Feature Engineering
- Filled Missing Values
- Gave Number based on Facility 

### 🔹 Feature Selection
- Removed irrelevant and redundant features
- Selected high-impact features for better performance

### 🔹 Outlier Detection & Removal
- Applied domain-based filtering
- Removed unrealistic data points
- Improved model accuracy
